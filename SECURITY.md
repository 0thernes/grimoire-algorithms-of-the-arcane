# Security Policy

GRIMOIRE is a static educational browser app. It has no backend, database, user accounts, secrets, analytics, service worker, or runtime API calls.

## Supported Surface

- Static HTML/CSS/JavaScript runtime.
- Local Web Audio synthesis.
- Markdown/JSON documentation.
- GitHub Pages workflow.

## Reportable Issues

Please report:

- unexpected external network requests
- unsafe links or root-relative paths that break project Pages
- script injection or content injection risk
- GitHub Actions workflow risk
- license/notice removal in forks or derivative distributions
- cryptographic/security records that overclaim production readiness

## Non-Claims

This project is not a cryptographic library, security control, package registry, package manager, or production algorithm suite. Algorithm cards are explanatory atlas records unless a future implementation cell explicitly contains real tested code.
