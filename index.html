TIER I — THE CLASSICAL RITES
Quantum-InspiredGenerative

Wave Function Collapse

Superposition collapsed into inevitability. Each cell exists in all possible states simultaneously until observed — then the wave collapses outward, propagating constraints like quantum entanglement through a tile grid. The universe builds itself by process of elimination.

MLGenerative

Diffusion

Signal buried in noise, noise burned away by signal. Forward: annihilate order into chaos. Reverse: learn the score function of reality and walk backward through entropy. A neural network learns to denoise — and in doing so, learns to dream.

OptimizationPhysics-Inspired

Simulated Annealing

Metallurgy encoded in math. Cool too fast and you get a brittle local minimum. Cool slowly and you find the global truth. The algorithm deliberately accepts worse solutions — a controlled chaos that burns away false summits to find the one true peak.

ChaoticSort

Sleep Sort

Each element is assigned a thread that sleeps for n milliseconds. Time itself does the sorting. The smallest values wake first and announce themselves. O(max(n)) — technically "correct" if the OS scheduler cooperates. Absurd. Genius. Works.

ChaoticSort

BogoSort

Is it sorted? No? Shuffle it. Repeat. Expected O(n·n!) — infinite patience required. The anthropic algorithm: given infinite time and infinite universes, one will be sorted by cosmic accident. This is that universe.

CryptographyNumber Theory

RSA

The trapdoor at the heart of civilization. Multiply two primes: trivial. Factor the product: computationally intractable. Your banking, your identity, your secrets — all protected by the asymmetry between multiplication and factorization. A mathematical one-way door.

QuantumCryptography-Killer

Shor's Algorithm

RSA's apocalypse, encoded in quantum gates. QFT finds the period of modular exponentiation in superposition — all possible periods evaluated simultaneously. When quantum computers scale, this algorithm ends the current era of cryptography. The door has a lock pick.

Geometry3D

Marching Cubes

Scalar fields turned into surfaces by 256 cube configurations. Every voxel corner is above or below the isosurface threshold — the 8-bit lookup table defines which triangles to draw. Terrain generation, medical imaging, fluid simulation: all march these cubes.

DistributedConsensus

Byzantine Generals / PBFT

How do you reach consensus when some messengers lie? The generals must agree on attack or retreat — but traitors send contradictory orders. PBFT guarantees consensus if fewer than ⅓ of nodes are malicious. Blockchain, distributed databases, the foundation of trustless systems.

StringSearch

Boyer-Moore

Reads right-to-left and skips entire sections of text on mismatch. The bad character and good suffix heuristics let it skip over provably impossible positions. Sub-linear in practice — it reads less text than exists. The best general-purpose string search algorithm ever devised.

TIER II — THE FORBIDDEN TEXTS
CursedSort

Stooge Sort

Recursively sort the first ⅔, then the last ⅔, then the first ⅔ again. O(n^2.709...) — slower than bubble sort, theoretically sound, deeply wrong. A sorting algorithm that works through sheer obstinate repetition. Named after The Three Stooges. Nyuk nyuk nyuk.

QuantumApocalyptic

Quantum Bogosort

Shuffle the array. Check if it's sorted. If not, destroy the universe. By the anthropic principle, you will only ever observe a sorted result — the unsorted universes cease to exist. O(1) in the surviving timeline. Requires many-worlds interpretation. Side effects: genocide of alternate realities.

SortOptimal Writes

Cycle Sort

Minimizes writes to O(n) — the theoretically minimum possible for a comparison sort. Finds the cycle that each element belongs to and rotates it into position. Critical for flash memory and write-expensive storage where every write costs lifespan.

StatisticsStreaming

Reservoir Sampling

Sample k items uniformly from a stream of unknown size, seeing each item once. The first k fill the reservoir. For each subsequent item i, replace a random reservoir element with probability k/i. Provably uniform. No second pass. Unknown n. Mathematically beautiful.

Exact CoverBacktracking

Algorithm X

Knuth's exact-cover ritual: choose the constraint with the fewest remaining choices, try a compatible row, cover every column it satisfies, recurse, and backtrack when contradiction appears. Sudoku, polyomino tiling, N-queens: all reduced to a sparse matrix of yes/no obligations.

Exact CoverBacktracking

Dancing Links (DLX)

The low-level engine beneath Algorithm X: a doubly linked circular matrix where rows and columns disappear in O(1), then reappear perfectly during backtracking. The links literally dance, letting a combinatorial search undo itself without rebuilding the world.

DistributedHashing

Jump Consistency Hashing

5 lines of code. Maps keys to buckets with perfect uniformity, minimal remapping when buckets change, and O(ln n) time. Uses a PRNG to simulate which bucket a key would "jump" to if the cluster grew. The entire algorithm fits in a tweet. Google runs this in production.

SortVibes

Cocktail Shaker Sort

Bidirectional bubble sort. Bubbles percolate up from left, then turtles crawl right from the right. The shaking motion addresses the "turtle problem" where small values near the end move agonizingly slowly leftward. Still O(n²) but feels more sophisticated at parties.

SortProduction

Timsort

Python's sort. Java's sort. Android's sort. Hybrid merge+insertion that exploits existing runs of sorted data in real-world arrays. Galloping mode shifts to exponential search when one run dominates. Adaptive, stable, O(n log n) worst case, O(n) best case. The most practical sort ever engineered.

Black Magicid Software

Fast Inverse Square Root

0x5F3759DF. One bit-level reinterpretation of a float as an integer, one subtraction from a magic constant, one Newton-Raphson iteration. Returns 1/√x accurate to 1%. Used in Quake III for real-time lighting calculations. The magic number is still not fully explained. Someone just tried it and it worked.

TIER III — THE DIMENSIONAL FUCKERY
SearchHeuristic

A* Search

Shortest-path divination with a compass. Dijkstra explores everything equally; A* adds a heuristic whisper toward the goal. The open set becomes a probability-looking frontier, but the guarantee remains hard: with an admissible heuristic, the first path found is optimal.

ProceduralNoise

Perlin Noise

Randomness with memory. Instead of static chaos, gradients blend into coherent fields: clouds, terrain, fire, cave veins, alien skin. It is noise that remembers where it came from, so the fake world feels like it obeys hidden geology.

BayesianSignal

Kalman Filter

A machine for turning noisy lies into a usable estimate. Prediction says where reality should be; measurement says where reality seems to be; uncertainty decides who gets trusted. Missiles, robots, satellites, AR tracking: all drink from this oracle.

ProbabilisticCardinality

HyperLogLog

Count billions of unique things with a handful of registers. Hashes that begin with many zeros are rare, so the longest zero streak estimates the crowd. It is census-taking by omens: tiny memory, shocking accuracy, no list of the actual souls.

HashingProbabilistic

Bloom Filter

A memory-efficient maybe-machine. Ask if an item exists: no means impossible, yes means maybe. Multiple hashes tattoo a bit array, accepting false positives to banish false negatives. Caches, databases, crawlers: all love this suspicious little oracle.

CryptoIntegrity

Merkle Tree

Tamper-evidence by recursive commitment. Hash every leaf, hash pairs into branches, hash branches into a root. Change one byte and the corruption climbs the tree like lightning. Blockchains, package managers, filesystems: all use this truth spine.

PlanningMonte Carlo

Monte Carlo Tree Search

Think by hallucinating futures. Selection follows confidence bounds, expansion opens a new branch, simulation rolls dice to the end, backpropagation teaches the ancestors. AlphaGo made it famous; game trees still hear it whisper.

EvolutionarySearch

Genetic Algorithm

Optimization by artificial evolution. Keep the fittest strings, splice them together, mutate the offspring, repeat until nonsense becomes target. It is not smart in the human way; it is a patient pressure gradient wearing Darwin's robe.

GenerativeFractal

L-System

A grammar that grows. Replace symbols with symbols, then interpret the sentence as turtle movement. Ferns, trees, lightning, blood vessels: recursive language becomes organic geometry. Tiny rule, vast botanical hallucination.

N-BodyQuadtree

Barnes-Hut Quadtree

Gravity without all-pairs madness. Nearby bodies are handled individually; faraway clusters collapse into one center of mass. O(n²) becomes O(n log n). A universe cheats by summarizing distant stars, and the simulation survives.
